# Install Friday

### 1. Friday发布在yoyi私有的npm中，安装Friday之前，需要把npm的上游指向私有地址
⚠️ 注意！在这里我们使用的是npm，因为我们使用npx来创建项目，所以需要修改npm上游地址

⚠️ 注意！<code>npx</code>需要<code>npm v5.2.0</code>版本以上

```css dark
npm config set registry http://registry.npm.yoyi.com.cn/
```
通过<code>npm config get registry</code>验证是否地址指向正确

### 2. 创建新应用，通过自定义的cra模版，快速创建一个friday项目
<code>@friday/library</code>包，就是我们创建的项目源


```css dark
npx create-react-app --scripts-version 4.0.0-next.64 [project-name] --template @friday/library
```
`由于目前cra4.03版本对tsconfig.json的限制导致新建模版中断，目前暂时使用4.0.0版本`
### 3. 启动项目
```css dark
yarn start 

```
几秒钟后，你会看到以下输出： 

```css dark
Compiled successfully!

You can now view friday-app in the browser.

  Local:            http://localhost:3000
  On Your Network:  http://192.168.7.35:3000

Note that the development build is not optimized.
To create a production build, use yarn build.

```
在浏览器里打开 http://localhost:3000 ，你会看到 Friday 的欢迎界面。

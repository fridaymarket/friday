# @friday/micro

微服务模块，基于qiankun。

https://github.com/umijs/qiankun



## `inMicroservice(): boolen` 是否在微服务框架中

```css dark

import { inMicroservice } from '@friday/micro'

inMicroservice()

```

## `DevelopmentLogin: React.FC<any>` 通用的微服务登陆页

应用在微服务框架中，一般会是用master登陆页，而在开发或者独立运行时，需要一个独立的登陆页面
```css dark

import { DevelopmentLogin } from '@friday/micro'


interface DevelopmentLogin {
	title?: string
	onSumbit(username: string, password: string): void
}

<DevelopmentLogin 
	title: document.title || '微服务通用登陆页面',
	onSumbit={(username, password) => {

	}}
/>

```

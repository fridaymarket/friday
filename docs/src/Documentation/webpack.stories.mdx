# @friday/micro-webpack-plugin

`@friday/micro-webpack-plugin` 目前封装了微服务需要的一些webpack配置，并且是在`customize-cra`配置的基础上。



## `devServerCorsOrHeader(headers = defaultHeaders)` 


配置webpack dev server的请求头.

```css dark
const defaultHeaders = {
	"Access-Control-Allow-Origin": "*"
}

devServer: overrideDevServer(
	// 添加cors
	devServerCorsOrHeader(),
)

```

## OverrideMicroOutPut(packageName)
重写output，支持微服务slave暴露生命周期方法入口

```css dark

const packageName = require('./package.json').name

webpack: override(
	OverrideMicroOutPut(packageName)
)
```
